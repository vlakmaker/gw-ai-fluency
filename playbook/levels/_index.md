# Levels

The playbook is organized into three levels. Each one builds on the last, so you can grow at your own pace.

- [[levels/basic|Basic]] — Foundational skills and first interactions with AI
- [[levels/intermediate|Intermediate]] — Applied skills and workflow integration
- [[levels/advanced|Advanced]] — Strategic use and system-level thinking

Start wherever feels right. If you're new to AI, [[levels/basic|Basic]] is a great place to begin. If you already use AI tools daily, you might jump straight to [[levels/intermediate|Intermediate]].
