# Levels

The playbook is organized into three levels. Each one builds on the last, so you can grow at your own pace.

- [Basic](/levels/basic/) — Foundational skills and first interactions with AI
- [Intermediate](/levels/intermediate/) — Applied skills and workflow integration
- [Advanced](/levels/advanced/) — Strategic use and system-level thinking

Start wherever feels right. If you're new to AI, [Basic](/levels/basic/) is a great place to begin. If you already use AI tools daily, you might jump straight to [Intermediate](/levels/intermediate/).
