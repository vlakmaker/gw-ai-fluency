# Workflow Automation

## Overview

Building repeatable, AI-assisted processes that save time and reduce manual effort. This pillar focuses on identifying automation opportunities, designing workflows, and integrating AI into existing processes.

**Community average score: 65%** — Solid middle ground. Most users are past basic, approaching intermediate.

## Three Levels

### 🟢 Basic

Identifying repetitive tasks and using AI to handle single-step automation (e.g., templates, text expansion, simple prompts).

### 🟡 Intermediate

Chaining multiple AI steps together, building reusable prompt sequences, and integrating AI into existing tools.

### 🔴 Advanced

Designing end-to-end automated workflows, building custom tools, and optimizing for reliability and edge cases.

## Learning Style Fit

- **Natural fit:** [Tinkerers](/archetypes/tinkerer/) and [Planners](/archetypes/planner/) — building processes rewards both experimentation and structure
- **Growth opportunity:** [Strategists](/archetypes/strategist/) — getting hands-on with automation turns strategic thinking into concrete results
